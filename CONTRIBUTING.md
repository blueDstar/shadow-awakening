# Contributing to Shadow Awakening

Thank you for your interest in contributing to **Shadow Awakening**! We welcome all contributions, from bug fixes and documentation to new features and dark-fantasy-themed quest ideas.

## How to Contribute

### 1. Reporting Bugs
- Use the **GitHub Issues** to report bugs.
- Provide a clear title and description, with as much relevant information as possible.
- Include steps to reproduce the issue.

### 2. Suggesting Enhancements
- Feature requests are also tracked in **GitHub Issues**.
- Explain the logic behind your suggestion and how it fits the "Shadow Awakening" theme (productivity, gamification, dark fantasy).

### 3. Pull Requests (PRs)
- Fork the repository and create your branch from `main`.
- If you've added code that should be tested, add tests.
- Ensure the app builds and runs locally before submitting.
- Write clear commit messages.

## Development Setup

### Backend (FastAPI)
- Navigate to `/backend`.
- Create a virtual environment: `python -m venv venv`.
- Install dependencies: `pip install -r requirements.txt`.

### Frontend (React/Vite)
- Navigate to `/frontend`.
- Install dependencies: `npm install`.
- Run dev server: `npm run dev`.

## Coding Style
- **Python**: Follow PEP 8.
- **JavaScript/React**: Use functional components and hooks. Maintain the Dark Fantasy UI aesthetic.
